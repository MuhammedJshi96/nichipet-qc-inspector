# Nichipet QC Inspector

Local Streamlit app for Nichiryo Nichipet EX II volume measurement / inspection.

## Stack
- Python 3.11
- Streamlit
- SQLite
- SQLAlchemy ORM
- Pydantic
- Pandas / NumPy
- pytest
- ReportLab

## Run
1. Create venv
2. Install requirements
3. Run `streamlit run app.py`

## Test
Run `pytest -q`
